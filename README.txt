Deep Dark Dungeon
             by Neprim

Управление:

На карте:
    wasd - перемещение по карте
    i - открыть/закрыть инвентарь
    l - осмотреться
    z - отмена действия
    > - спуститься вниз по лестнице
    < - подняться вверх по лестнице
    , - поднять предметы
    Пробел - пропустить ход

В инвентаре:
    [a-z] - выбор предмета
    E - переключение между снаряжением/инвентарём
